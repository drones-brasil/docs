## Fotos (fonte)

Coloque PNG/JPG/WebP aqui com nome `<id>.<ext>`. No build vira **WebP (máx. 256px)**:

- `content/photos/authors/<authorId>.*` → `/img/authors/<authorId>.webp`
- `content/photos/teams/<teamId>.*` → `/img/teams/<teamId>.webp`
- `content/photos/universities/<universityId>.*` → `/img/universities/<universityId>.webp`
